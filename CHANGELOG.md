# 更新日志

## v0.2.0 (2026-05-13)

### ✨ 新功能

- **富文本编辑器工具栏大升级**：
  - 新增 **下划线** 按钮
  - 新增 **标题下拉选择**（H1 / H2 / H3 / 正文），替换原来的无效 H 按钮
  - 新增 **引用块** 按钮
  - 新增 **左/中/右对齐** 按钮
  - 新增 **插入链接** 按钮（弹窗输入 URL）
  - 新增 **分割线** 按钮
  - 新增 **清除格式** 按钮
- 正式加入 **electron-builder 打包配置**（NSIS 安装包 x64）

- 修复"标题"按钮调用 `execCommand('heading')` 无效的问题，改用 `formatBlock` + 下拉选择器

### 🏗️ 基础设施

- 新增 `npm run pack` 打包脚本
- 新增 `package.json` 的 `build` 配置段
- 新增 CHANGELOG.md 版本记录
- 更新 README.md 项目说明
